# SQL-Product-Analysis
Designed a relational database and wrote analytical SQL queries to generate insights on product sales, orders, and revenue.
This project demonstrates SQL-based analysis of product sales and orders using PostgreSQL.

## Project Overview
The database stores products and customer orders. Analytical SQL queries are used to generate business insights such as revenue, order trends, and product performance.

## Database Structure

### Products Table
- product_id
- product_name
- category
- price
- stock_quantity
- is_available
- added_on

### Orders Table
- order_id
- product_id
- quantity
- order_date
- customer_name
- payment_method

##  How to Run

1. Create a database in PostgreSQL.
2. Run the SQL script to create tables.
3. Import CSV datasets.
4. Execute analysis queries.
